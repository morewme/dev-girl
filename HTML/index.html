O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo de páginas web. É o "esqueleto" de todo site — define o que é um título, um parágrafo, uma imagem, um link, etc, por meio de tags.

Um exemplo básico de documento HTML é o seguinte:

  
    <html lang="pt-BR">
      <head>
        <meta charset="UTF-8">
        <title>Minha Página</title>
      </head>
      <body>
        <h1>Olá, mundo!</h1>
        <p>Este é um parágrafo de texto.</p>
        <a href="https://exemplo.com">Clique aqui</a>
      </body>
    </html>

A principal característica do HTML é a utilização de tags para definir a estrutura e o conteúdo das páginas web. Tags são elementos que delimitam o início e o fim de uma estrutura, como <h1> para títulos e <p> para parágrafos.

Todo documento HTML está dentro de uma tag <html>.
