O Flexbox (Flexible Box Layout) é um modelo de layout do CSS criado para distribuir e alinhar elementos dentro de um contêiner de forma dinâmica e eficiente.

A ideia central é simples: você tem um contêiner pai (o flex container) e filhos (os flex items). Quando você define display: flex no pai, todos os filhos imediatos passam a se comportar de acordo com as regras do Flexbox.

Existem dois eixos fundamentais que controlam tudo:

  • Eixo principal (main axis): a direção em que os itens são dispostos (horizontal por padrão)
  • Eixo cruzado (cross axis): perpendicular ao eixo principal

Propriedades do contêiner (pai):

  • flex-direction: define a direção dos itens no contêiner
  • flex-wrap: controla se os itens devem quebrar para a próxima linha
  • justify-content: alinha os itens no eixo principal
  • align-items: alinha os itens no eixo cruzado

Propriedades dos itens (filhos):

  • order: controla a ordem de exibição dos itens
  • flex-grow: define a capacidade de um item crescer para preencher o espaço disponível
  • flex-shrink: define a capacidade de um item encolher quando o espaço é limitado
  • flex-basis: define o tamanho inicial do item antes de crescer ou encolher
