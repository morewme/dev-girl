Bem-vindo ao CSS!

O que é CSS?

O CSS (Cascading Style Sheets) é a linguagem responsável pela aparência visual das páginas web. Se o HTML é o esqueleto, o CSS é a roupa e a maquiagem.

Como funciona

O CSS funciona selecionando elementos HTML e aplicando estilos a eles:

p {
color: red;
font-size: 18px;
}

Esse código deixa todos os parágrafos com texto vermelho e tamanho 18px.

Onde colocar o CSS?

O CSS pode ser colocado de três formas:

  • Inline: diretamente no elemento HTML usando o atributo "style".
    • <p style="color: red;">Texto vermelho</p>
  • Interno: dentro de uma tag <style> no cabeçalho do documento HTML.
    • <style>
    p {
    color: red;
    font-size: 18px;
    }
    </style>
  • Externo: em um arquivo .css separado, que é vinculado ao HTML usando a tag <link>.
    • <link rel="stylesheet" href="estilo.css">

O que você pode estilizar

  • Cores e fundos — color, background-color, background-image
  • Tipografia — font-family, font-size, font-weight, line-height
  • Espaçamento — margin (espaçamento externo) e padding (espaçamento interno)
  • Tamanho — width, height, max-width
  • Layout — display, flexbox, grid
  • Bordas e sombras — border, border-radius, box-shadow
  • Animações — transition, animation

A Cascata no CSS

Quando dois estilos conflitam, o navegador resolve o impasse seguindo três critérios em ordem:

  • Especificidade — a regra mais "específica" vence. Um ID (#titulo) pesa mais que uma classe (.texto), que pesa mais que um elemento (p). Quanto mais direcionado o seletor, maior sua autoridade.
  • Ordem de aparição — se dois seletores têm o mesmo peso, vence o que aparece por último no código. O CSS lê de cima para baixo, e a última palavra é a que vale.
  • Herança — elementos filhos herdam automaticamente certas propriedades do pai, como color e font-family. Propriedades de espaçamento e borda, porém, não são herdadas.

Esses três critérios formam a "cascata" — um sistema de prioridades que garante previsibilidade mesmo quando múltiplas regras se aplicam ao mesmo elemento.
